Get Reel
====

> Big fish reelin'.

### Running locally

Make sure you have `node 0.11.x` installed, as [Koa](http://koajs.com/) requires generators.

Also make sure you have [ngrok](https://ngrok.com/) installed.

1. Run `make`
2. Run `ngrok 3000`
3. Open your browser to `localhost:3000`
4. Open your mobile browser to whatever the ngrok address is
5. ???
6. Profit
